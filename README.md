# site infosec iniciante

## Sobre

O site foi criado para documentar meus estudos de infosec.

## Tecnologias

- Node.js → backend
- Pug → frontend
- Render → hospedagem gratuita

## Como rodar

- Na raiz do projeto rode no terminal:

```bash
//baixar dependências
$ npm install

//rodar o projeto
$ npm start
```
